# Kundservice Chatbot AI

Final project for the Building AI course

## Summary
Min idé är en AI-baserad chatbot som kan hantera kundservice automatiskt genom att svara på vanliga frågor och hjälpa kunder dygnet runt. Den ska kunna ge snabba och korrekta svar utan att kunden behöver vänta på en människa.

## Background
Många företag får stora mängder kundförfrågningar varje dag, vilket kan leda till långa väntetider och stress för personalen. Jag valde detta problem eftersom bra kundservice är avgörande för kundnöjdhet, men samtidigt resurskrävande. Det finns därför ett tydligt behov av smarta lösningar som kan avlasta personalen.

## How is it used?
Chatboten används via företagets hemsida eller app. Den riktar sig till kunder som behöver snabb hjälp, och till företag som vill effektivisera sin kundservice. Användaren skriver sin fråga och får direkt ett svar från systemet.

## Data sources and AI techniques
Systemet använder historisk kunddata, tidigare frågor och svar samt textdata från chattar. AI-tekniker som naturlig språkbehandling (NLP) och maskininlärning används för att förstå frågor och ge relevanta svar. Genom att analysera stora mängder data kan systemet hela tiden bli bättre.

## Challenges
En utmaning är att chatboten inte alltid förstår komplexa eller ovanliga frågor. Det finns också risk för att kunder blir frustrerade om svaren känns opersonliga. Integritet och hantering av kunddata är också viktigt. Därför behöver systemet kontinuerligt förbättras och övervakas.

## What next
I framtiden kan chatboten bli mer avancerad genom att förstå känslor (t.ex. om kunden är frustrerad) och samarbeta bättre med mänsklig personal. Den skulle också kunna hantera mer komplexa ärenden. På så sätt kan den bli en ännu viktigare del av företagets kundservice.
